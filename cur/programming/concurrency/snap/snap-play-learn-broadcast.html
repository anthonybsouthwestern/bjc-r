While you may have already used multiple sprites, now we will see how you can have them interact with each other via message passing.

The broadcast block allows us to control which block is run next through message passing. Try to make these scripts in Snap!.

  • You'll need to add a sprite to your project: use between the stage and the sprite corral, at the center right of the window.
  • Each sprite will have separate costumes and scripting areas: click on the image in the sprite corral to focus on it. To add costumes, you just need to drag an image from a page in your browser over to the scripting area of your snap window. Here are costumes you can use:
  • Each sprite will have different scripts. To switch between the scripts and costumes for a sprite, use the tabs at the top of the scripting area.
  • Once you are done, press to start the short play.

A note about style: You will notice that we chose to name the messages that were broadcast so that it would help us keep track of what we were doing and what messages we were sending. We recommend that you do this in your projects!
