The RELABEL block is a block we can use to rename an existing column. The RELABEL block is a command block, so it will permanently affect the Table you are working with; it will not make a copy.

The RELABEL block does not change the dimensions of the Table, but it changes a column name. It will still have the same number of rows and columns as before.

The RELABEL block takes in 3 arguments. These arguments are:

  1. An existing column name (not an index)
  2. The new column name
  3. A Table
