The last data visualization technique we will cover is a line plot. Line plots are used to visualize the distribuation of two numerical variables. Both discrete and continuous categorical variables can be visualized using a line plot. The numerical variable on the x-axis must be a variable that indicates the passing of time, and the second numerical variable can be aby variable leasured over that period of time.

The line plot block takes in 4 arguments, 2 that are required and 2 that are optional:

  1. A Table from which some data will be visualized using a line plot
  2. A column label whose values represent the passage of time
  3. A column label to use to group the first column (optional)
  4. A title for the plot (optional; will generate a default title if none is provided)

The first column label must contain time values that are evenly spaced out. Always make sure if you want a line plot that your time variable has a measurement for consistent intervals of time. The line plot block will error if this is not the case.
