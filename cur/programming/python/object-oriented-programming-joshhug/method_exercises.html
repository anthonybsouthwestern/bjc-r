  1. Exercise 1: Please complete the method outdated, which determines if a book has reached old age. The takes in a book object and an age, and if the book's age is greater than the age given, the book object should be assigned a new instance attribute "worn_down" which is set to True. 
    
>>>strange_book = Book("Cooking", "How to Cook Water", "Yaxson Mang", 2016)
>>>strange_book.calculate_age()
2
>>>strange_book.outdated(4)
>>>strange_book.worn_down
True

    Note that the example above is contrived. Methods like "outdated" that set a new instance variable as a side effect are generally undesirable. It's better design to simply have the outdated method return True rather than setting some other arbitrary value to True.

  2. Edit the Book class so that it has a new class attribute called varieties, initialized to an empty dictionary. Then, add a method add_genres to the Book class that takes in a book object. If the book's genre does not exist inside varieties, then we add the genre as a key, with its value being a list of lists. Each list inside this list of lists should contain a book's title and author. If the genre does exist already, then we simply add our book object's title and author to the corresponding genre's value. The same book title and author should not appear more than once for a given genre, even if that book object's method is called multiple times. 
    
>>>Book.varieties
{}
>>>old_book = Book("Comic", "Calvin & Hobbes", "Bill Watterson", 1985)
>>>Book.add_genres(old_book)
>>>Book.varieties
