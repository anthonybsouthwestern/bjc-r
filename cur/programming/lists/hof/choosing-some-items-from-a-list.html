You've might have played Geography, the game in which one player says a place name and the next player has to name another place whose first letter is the last letter of the previous place: Newark, Kentucky, Yonkers, San Francisco, Oklahoma. Have you ever gotten stuck on the letter "A"? Oklahoma, Alabama, Asia, Alaska, America, Alameda... and so on, forever.
