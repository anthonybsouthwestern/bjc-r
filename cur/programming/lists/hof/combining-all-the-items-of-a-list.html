Create, test, comment on, and submit reporters for all of the following functions used in data science and/or statistics:

  • sum of: the sum of all of the values in the list
  • average of: the average of the values in the list, using the sum of block
  • max of: the largest value in the list
  • min of: the smallest value in the list
  • a predicate increasing: reports true if each consecutive value is at least as large as the previous value
  • median of: the middle value of an odd number of items or the average of the two middle values for an even number of items. For now, check that the list is increasing first, and if it is not, simply report that rather than calculating the median.
  • mode of: the most common value in the list. For now, check that the list is increasing first, and if it is not, simply report that rather than calculating the mode.
